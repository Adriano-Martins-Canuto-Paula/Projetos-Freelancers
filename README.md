# Projetos Freelancers

Este repositório reúne **projetos desenvolvidos como freelancer**, com foco em soluções digitais aplicadas a **necessidades reais de clientes**. Os trabalhos aqui apresentados envolvem principalmente **desenvolvimento web**, com atenção à identidade visual, usabilidade e objetivos comerciais de cada projeto.

Cada projeto possui uma descrição resumida, explicando o **contexto**, o **objetivo** e a **solução entregue**, servindo como portfólio técnico e profissional.

---

## Projetos

### 🔹 Landing Page — Ebenezer

Projeto de **landing page institucional** desenvolvido para a empresa **Ebenezer**, com o objetivo de **melhorar a visibilidade web da marca** e fortalecer sua presença digital.

A página foi pensada para apresentar de forma clara:
- A identidade da empresa
- Seus produtos e serviços
- Endereço físico
- Formas de contato

O principal objetivo do projeto foi **atrair o público online** e **converter visitantes em potenciais clientes**, direcionando-os para os canais de contato da empresa, facilitando o processo de compra e comunicação.

**Tipo:** Landing Page  
**Foco:** Visibilidade online e conversão  
**Status:** Concluído
